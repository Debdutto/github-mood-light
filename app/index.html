About


This website tracks events happening across GitHub and converts them to soothing lights and music.


Four types of events are tracked: PushEvent, PullRequestEvent, IssuesEvent and IssueCommentEvent. The colors are generated based on three knob values each for Red, Green And Blue. Each knob is varied depending on the type of event and the length of the payload. There are three types of sounds. Bells represent PushEvents, string plucks represent IssuesEvents and IssueCommentEvents whereas string swells represent PullRequestEvents.
